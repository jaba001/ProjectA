# ProjectA

Unreal Engine 기반 **Grid Turn-Based Combat System** 프로젝트입니다.  
*For The King* 스타일 전투 시스템을 목표로 설계되었습니다.

## Features

- Grid 기반 전투 시스템
- Turn Manager 기반 턴 진행 구조
- Gameplay Ability System(GAS) 활용
- Tile 기반 이동 및 타겟 선택
- Unit AI Controller

## Tech Stack

- Unreal Engine 5
- C++
- Gameplay Ability System (GAS)

## Project Structure
Source/ProjectA
ㄴGrid // CombatGridTile
ㄴUnit // UnitBase
ㄴTurn // TurnManager
ㄴGAS // Ability / Effect
ㄴPlayer // PlayerController

## Goal

GAS를 활용한 **확장 가능한 턴제 전투 아키텍처 설계**를 목표로 합니다.
